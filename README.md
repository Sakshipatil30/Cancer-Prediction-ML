# Cancer Prediction Using Machine Learning

## Project Overview
This project predicts whether a tumor is malignant or benign using machine learning techniques.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

## Model Used
Gradient Boosting Classifier

## Accuracy
Model achieved approximately 96-98% accuracy.

## Conclusion
Machine learning models can assist in early detection of breast cancer and improve medical decision-making.# Cancer-Prediction-ML
